AirportWeb API

Powerful aviation management API for modern applications

API Overview

The AirportWeb API provides a comprehensive set of endpoints for managing airport operations, user accounts, flight information, bookings, and more. This RESTful API enables developers to build robust aviation management applications with secure authentication, user management, and flight operations.

Secure Authentication

JWT-based authentication with role-based access control for all endpoints

Flight Management

Comprehensive flight operations including search, details, and status tracking

Booking System

End-to-end booking management with payment processing integration

Travel Analytics

User travel statistics and preferences for personalized experiences

Authentication

POST /register

Create a new user account with email and password.

Request Body

email User's email address (must be unique)
password Strong password (min 8 characters)
POST /login

Authenticate a user and return a JWT access token.

Request Body

email Registered email address
password Account password

User Management

POST /regUserDetail

Register additional details for a user profile.

Request Body

userId ID of the user to update
details Object containing user details (name, phone, address, etc.)
GET /getUserDetail/:id

Retrieve detailed information for a specific user.

URL Parameters

id ID of the user to retrieve

Preferences & Statistics

POST /userpreferences

Set or update user preferences for travel experience.

Request Body

userId ID of the user
preferences Object containing user preferences (seat, meal, notifications, etc.)
GET /userpreferences/:userId

Retrieve preferences for a specific user.

URL Parameters

userId ID of the user
POST /travellingStat/:id

Add travel statistics for a user.

URL Parameters

id User ID

Request Body

stats Travel statistics object (miles, flights, destinations, etc.)
GET /travellingStats/:id

Retrieve travel statistics for a specific user.

URL Parameters

id User ID

Flight Operations

GET /flightDetails

Retrieve a list of all available flight details.

Query Parameters

limit Number of results to return (optional)
offset Pagination offset (optional)
POST /flightSearch

Search for flights based on criteria.

Request Body

origin Departure airport code
destination Arrival airport code
date Departure date (YYYY-MM-DD)
passengers Number of passengers (optional)

Booking & Payment

POST /booking

Create a new flight booking.

Request Body

userId ID of the user making the booking
flightId ID of the flight to book
passengers Array of passenger details
GET /booking/:id

Retrieve booking details for a specific user.

URL Parameters

id User ID
POST /payment

Process payment for a booking.

Request Body

bookingId ID of the booking to pay for
paymentMethod Payment details (card, PayPal, etc.)
GET /payment

Retrieve payment details (admin only).

Query Parameters

bookingId Filter by booking ID (optional)
userId Filter by user ID (optional)

Admin Endpoints

POST /flightDetails

Create new flight details (admin only).

Request Body

flightNumber Unique flight number
origin Departure airport code
destination Arrival airport code
departure Scheduled departure datetime
arrival Scheduled arrival datetime
PUT /flightDetails/:id

Update existing flight details (admin only).

URL Parameters

id Flight ID to update

Request Body

updates Object containing fields to update
